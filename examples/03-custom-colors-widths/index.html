EU Energy Label - Custom colors and widths

You can customize bar colors and widths overriding the CSS:

energy-label .energy-class:nth-child([n]) .energy-class-label {
  background: [width];
}

energy-label .energy-class:nth-child([n]):not(.energy-class-disabled)
.energy-class-label {
  background: [color];
}

energy-label .energy-class:nth-child([n]):not(.energy-class-disabled)
.energy-class-label:not(.disabled)::after {
  border-left-color: [color];
}

Where [n] is the 1-based index of the bar, [color] a valid CSS color definition, [width] a valid CSS width.
