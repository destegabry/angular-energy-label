EU Energy Label - Using SASS

To the Energy Labels with SASS you have to install Bourbon in your project (npm install bourbon) and import it strong angular-energy-labels.scss.

You can change the default Energy Label rendering setting some values before importing angular-energy-labels.scss. 

@import 'node_modules/bourbon/app/assets/stylesheets/bourbon';

$energy-label-class-label-count: 11;
$energy-label-class-label-colors: #c5cae9 #9fa8da #7986cb
                                  #5c6bc0 #3f51b5 #3949ab
                                  #303f9f #283593 #1a237e
                                  #121e61 #020933;
$energy-label-class-label-disabled: #f3c1c1;
$energy-label-class-label-width: 15%;
$energy-label-class-label-add-width: 3%;

@import 'node_modules/angular-energy-label/angular-energy-label.scss';

Refer to the source code for an exhaustive list of all the supported default values.
